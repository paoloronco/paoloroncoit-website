Managing multiple projects and websites is a testament to my skills and passion for technology. To efficiently protect and manage these sites, I use Cloudflare Zero Trust, a service that offers a wide range of solutions to enhance web security and performance. Cloudflare Zero Trust protects my sites from attacks and threats, allowing me to expose them securely on the Internet via self-hosting on Proxmox servers. Below is a complete overview of my websites, with details on their function and configuration.

#### MAIN WEBSITE

**URL:** [https://www.paoloronco.it/](https://www.paoloronco.it/)  
**Technology:** Cloudflare DNS, Aruba Hosting  
**Description:** This is my main site where you can find information about me, my CV, and contact details. Currently hosted on Aruba and using Cloudflare for DNS management. It will likely be migrated to self-hosting by 2025.

#### PORTFOLIO

**URL:** [https://prportfolio.paoloronco.it/](https://prportfolio.paoloronco.it/)  
**Technology:** Self-hosting on Proxmox, Docker, Docker Compose, Portainer, Cloudflare Zero Trust  
**Description:** A dedicated site for publishing articles about new software, services, apps, and experiences I have developed. It uses Docker with Docker Compose and Portainer for application management, exposed securely via Cloudflare Zero Trust. WordPress is installed on Portainer for content management.

**Main Website**

URL: [https://portfolioronco.it](https://portfolioronco.it) and [https://portfolioronco.com](https://portfolioronco.com)  
Description: These domains act as redirects to the main portfolio site, making the domain name simpler and more accessible.

#### Link’s

URL: [https://links.paoloronco.it/](https://links.paoloronco.it/)  
**Technology:** Self-hosting on Proxmox, Docker, Docker Compose, Portainer, Cloudflare Zero Trust  
**Description:** An ordered index of all my sites. It uses LinkStack as a graphical solution, distributed via Portainer from GitHub. The site is exposed securely via Cloudflare Zero Trust.

#### LiveTerm for Linux Enthusiasts

URL: [https://paoloronco-liveterm-vercel.vercel.app/](https://paoloronco-liveterm-vercel.vercel.app/)
