[GitHub Repository: GamesManager-WebApp](https://github.com/paoloronco/GamesManager-WebApp)

In my portfolio website, one of the projects I want to showcase is my GitHub repository [GamesManager-WebApp](https://github.com/paoloronco/GamesManager-WebApp). This project, entirely developed by me using HTML, CSS, and JavaScript, is a versatile web application designed to help manage and organize games efficiently.

#### Description

**GamesManager-WebApp** was created to simplify the management of games across multiple platforms and disks. The application allows you to track installed and non-installed games, offering useful tools for other types of collections or resources as well.

#### Features

-   **Complete Game Management:** Allows you to view and manage both installed and uninstalled games, tracking their location and platform.
-   **Platform Organization:** Enables monitoring of games on platforms such as Steam, Epic Games, EA, Ubisoft, etc.
-   **Disk Installation Management:** Lets you check where the games are installed, whether locally or remotely.
-   **Tracking Uninstalled Games:** Keeps a record of owned but uninstalled games, verifying their availability on various platforms.

#### Use Cases

GamesManager-WebApp is ideal for:

-   **Managing Games Across Multiple Platforms:** Easily track games across multiple platforms and disks.
-   **Organizing Collections:** Can be used for other collections that require tracking and management.

#### How to Use It

-   **Add Your Games:** Enter your games with details such as platform, installation status, and location.
-   **Filter and Search:** Use the search bar and filter buttons to quickly find your games.
-   **View Details:** Access game details through an organized and intuitive interface.

#### Interface

The app's interface includes:

-   **Search Bar:** For searching for games by name or other attributes.
-   **Filter Buttons:** To filter games by category, type, and location.
-   **Game Table:** Displays game details in a structured table:

| Name | Category | Type | Location |
| --- | --- | --- | --- |
| Item 1 | Category 1 | Type 1 | Local |
| Item 2 | Category 2 | Type 1 | Local |
| Item 3 | Category 3 | Type 2 | Remote |
| Item 4 | Category 4 | Type 2 | Remote |
| Item 5 | Category 1 | Type 1 | Local |
| Item 6 | Category 2 | Type 1 | Local |
| Item 7 | Category 3 | Type 2 | Remote |
