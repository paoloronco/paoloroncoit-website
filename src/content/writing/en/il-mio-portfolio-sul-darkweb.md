An online portfolio is an essential tool for getting known by potential clients and collaborators. However, it's not always easy to find a way to showcase one’s skills in a secure and accessible manner.

For this reason, I decided to create a WebServer to expose, in a safe manner, my website acting as a resume on the DarkWeb through TOR.

The DarkWeb is a part of the Internet that is accessible only via specific software, such as Tor browsers. This makes the DarkWeb a safer place to showcase one’s work, as it is harder to monitor by third parties.

My website on the DarkWeb presents my most recent works, along with a brief biography and a contact form. I paid particular attention to the security of the site, using a series of measures to protect data and sensitive information.

Firstly, the site is hosted on a secure server with a firewall and data encryption system. Additionally, the site uses an SSL certificate to ensure the security of data transmitted between the server and the visitor’s browser.

Furthermore, I implemented additional security measures such as two-factor authentication and multi-factor authentication systems. These measures make it harder for hackers to access the site or visitors' data.

## Some Tips for Creating a Secure Portfolio on the DarkWeb

If you are thinking about creating a portfolio on the DarkWeb, here are some tips to help you do so securely:

- **Choose a secure server.** The server hosting your site should be protected by firewalls and data encryption.
- **Use an SSL certificate.** An SSL certificate ensures the security of data transmitted between the server and the visitor’s browser.
- **Implement additional security measures.** Besides basic security measures, you can also implement further security measures such as two-factor authentication and multi-factor authentication systems.
- **Be aware of the risks.** Even though the DarkWeb is a safer place to showcase one’s work, it's important to be aware of the risks involved. For example, your site could be hacked or your data stolen.
